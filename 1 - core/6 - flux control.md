### Controle de fluxo

Condicionais.

```python
if (expression) 
    print("value 1");
elif (expression)
    print("value 2");
else
    print("value 3");
```

Loops.

```php
// Comum.
for (a = 0; a<100; a++)
    print("Hello world\n");

for (a = 0; ; a++) {
    if (a >= 100)
        break;
}

for (a = 0; ; a++) {
    if (a < 100)
        next;
}

// Sem incremento.
for (a=0; a<10) {
    print("Data...\n");
    a++;
}

// Equivalente ao while. Apenas condição.
for (a < 10)
    print("a: {}\n", a);

// Loop infinito.
for ()
    print("Infinite...\n");
```
