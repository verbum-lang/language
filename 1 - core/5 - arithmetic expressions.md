### Expressões aritméticas

Alguns exemplos.

```php
// Atribuição de valores à variáveis simples.
variable = (10 + (variable++) + (3.14 * 3) / --abc) - 10;

// Como elemento em arrays indexados.
variable = [
    1, 2, 3,
    (10 + ( 20 * 3 ) / 4),
    'str 1', "str 2"
];

// Como elemento em arrays associativos.
variable :array = [
    { name:  "Verbum" },
    {
        value: 'Divino',
        operation:  3 + ( 3 ) + 3
    }
];
```
